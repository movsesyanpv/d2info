Indeedstor (Дустанатор) - бот для Destiny 2

Indeedstor (Дустанатор) - многофункциональный бот по Destiny 2. Он предоставляет актуальную информацию о мире игры. С помощью бота можно строить клановые списки лидеров и организовывать сборы в активности.

Сервер Discord Добавить бота

Возможности

Информация о ресетах

После каждого ресета бот отправляет информацию о мире Destiny 2 в зарегистрированные каналы

Списки лидеров клана

Можно строить рейтинги членов клана по любым метрикам, которые можно надеть на эмблему в игре

Организация сборов

Создавайте сборы в активности. Для добавления в список достаточно нажать на соответствующую кнопку под сбором. Возможно ручное одобрение желающих

Слеш-команды

Бот полностью поддерживает слеш-команды

Многоязычность

Бот поддерживает те же языки, что и Destiny 2. Локализация выполнена пользователями бота
